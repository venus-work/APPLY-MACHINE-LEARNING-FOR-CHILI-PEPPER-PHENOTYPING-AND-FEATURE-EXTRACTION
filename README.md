## Installation

``` shell
# Clone github yolov7 to maim
git clone https://github.com/augmentedstartups/yolov7.git
cd yolov7

# Install requirement of yolov7
pip install -r requirements.txt
```
Move file object.py to utils and my_detect.py to main

    - object.py
